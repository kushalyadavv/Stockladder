## App details (full description)

Paste into the main listing description field:

---

**Stop manually reordering collections. StockLadder does it for you.**

Every hour your catalog changes: items sell out, new stock arrives, bestsellers shift. StockLadder automatically reorders products inside your Shopify collections so shoppers see in-stock, high-priority products first and sold-out SKUs at the bottom.

**Built for catalog-heavy stores** : fashion, beauty, supplements, multi-SKU brands, and any merchant tired of dragging products in the admin.

### What StockLadder does

- **Inventory-first sorting** : Products with more stock rank higher; out-of-stock items move to the bottom (or can be hidden entirely).
- **Sales-aware ranking** : Blend units sold or revenue from the last 7, 30, or 60 days with inventory levels.
- **Safe snapshots** : Every live sort saves the previous order. Preview with dry run, then revert in one click.
- **Per-collection rules** : Set global defaults, then override specific collections (skip sale pages, pin featured tags, demote clearance vendors).
- **Real-time webhooks** (Growth+) : Re-sort automatically when inventory changes.
- **A/B strategy compare** (Pro) : Preview two sort strategies side by side before applying.

### How to get started

1. Install StockLadder and approve permissions.
2. Open the app from Shopify Admin.
3. Choose a sort strategy and collections.
4. Run a **dry run** to preview, then **Run sort** when ready.

## Demo store / test instructions (for Shopify reviewer)

Include in **Testing instructions** field:

---

**Test store:** Use the development store provided during review, or install on any dev store linked to our Partner account.

**Steps to test:**

1. Install StockLadder from the app listing or open:
   `https://stockladder.xyz/auth?shop=YOUR-DEV-STORE.myshopify.com`
2. Approve OAuth permissions (read/write products, read inventory, read orders).
3. In the app, go to **Run & preview**.
4. Select a collection with multiple products (e.g. "All products" or a manual collection).
5. Click **Load preview** to see the proposed order.
6. Click **Dry run** to simulate a sort without writing changes.
7. Click **Run sort** to apply the new order live.
8. Open the collection in Shopify Admin → Products and confirm order changed.
9. Click **Revert snapshot** to restore the previous order.
10. (Optional) Go to **Plan & billing** and start a test subscription on the Growth plan (test charges on dev stores).

**Scopes used:**

- `read_products`, `write_products` : fetch collections and reorder products
- `read_inventory` : inventory levels for sorting
- `read_orders` : aggregate sales units/revenue for sales-based strategies (no customer PII accessed)

**Notes:**

- App is embedded in Shopify Admin.
- Billing uses Shopify's native subscription API (test mode on dev stores).
- No external login required beyond Shopify OAuth.
